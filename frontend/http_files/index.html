WASTT | a Web Application Security Testing Tool

This tool includes handy features like a proxy for traffic interception, a repeater for custom request testing, an intruder for automated attacks, comprehensive logging and reporting, and fuzzing tests to identify potential security vulnerabilities. It's designed to help ensure web applications are secure.
The tool was developed for a graduate project.
